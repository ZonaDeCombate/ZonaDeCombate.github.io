Bem vindo, soldado.

Seguindo o processo de melhorias no Zona de Combate, adicionamos um modo alternativo ao SquadSync para jogar em nosso servidor principal. Esperamos que desta maneira, todas as casas participem de nossas missões diárias.

Sem mais delongas, basta seguir os passos abaixo.


Fazendo o download do Swifty

O Swifty é um gerenciador de Mods especialmente desenvolvido para o Arma 3.
Utilizado por grandes grupos como o Shack Tactical e o Folk Arps, é uma ferramenta simples, mas extremamente poderosa, que consegue verificar seus mods e se necessário baixar qualquer atualização de nossos servidores, de maneira rápida e otimizada.

Acesse o site https://getswifty.net para baixar o Swifty

.

1- Clique no menu Download

2- Em seguida, clique no botão Download

Instale o aplicativo e execute-o para continuar este tutorial.

Na primeira vez que for executado, o Swifty informará que você pode configurar o programa.
É isso que faremos agora. Clique no botão OK.

3- Clique no ícone de Configurações

4- Arraste a barra de downloads em paralelo para 5

5- OPCIONAL: Clique em Browse se quiser determinar uma pasta temporária de downloads diferente da que ele propõe.

6- Clique em Browse e escolha a PASTA em que seu Arma 3 está localizado.
Certifique-se que a caixa Autocopy Teamspeak Plugins está habilitada, como na imagem abaixo.

7- Se você utiliza uma pasta de mods que não seja a raiz do Arma 3, clique em Add para determinar sua localização.

8- Em seguida, clique no botão Save para finalizar esta parte.

9- Clique no botão + para adicionarmos o servidor do Zona de Combate.

10- Clique no botão Import para adicionarmos o IP do Zona de Combate.

11- Copie o IP abaixo e cole no campo URL 

http://arsenal.zonadecombate.com.br:8080
12- Clique em Import para continuar.

13- OPCIONAL: Você pode trocar o nome aqui, se quiser.

14- Clique em Browse para selecionar a pasta onde seus mods se encontram.

15- OPCIONAL: Se você utiliza parâmetros para abrir o Arma, você pode inseri-los aqui.
Como utilizar estes parâmetros foge do escopo deste tutorial, mas você pode encontrar mais informações sobre eles aqui.

16- Clique no botão Save para continuar.

Em seguida, o Swifty iniciará o processo de verificação de Mods.
A mensagem de Checking table aparecerá.

Agora, o Swifty irá catalogar todos os seus mods, para futuras verificações.
Este processo é mais demorado e irá acontecer somente uma vez.
Tenha paciência e certifique-se de NÃO FECHAR O PROGRAMA.

17- Após feita a verificação, pode ser que você precise atualizar seus mods.
Para isso, clique no ícone de Update.

18- Para este tutorial, o mod MELB aparece como necessário para download, assim como seu tamanho.
Para você, os mods poderão ser diferentes.

Clique em Download para iniciar a sincronização de mods com o servidor do Zona de Combate.

O Swifty então inicia o download dos mods que você precisa para entrar no Zona de Combate.

19- Após baixar os mods necessários, clique em Recheck para uma última verificação.

20- Após a verificação final, coloque o ponteiro do mouse sobre o botão PLAY.

21- Em seguida, clique em Join para entrar no servidor!

OPCIONAL: Utilizando mods externos

O servidor Zona de Combate aceita alguns mods que melhoram a experiência do jogador mas são considerados muito pessoais para serem colocados na lista principal.
Esta lista não é final e pode mudar de acordo com sugestões da comunidade. São eles:

Blastcore: Phoenix 2
JSRS3: DragonFyre
DynaSound


Neste caso, o Swifty permite que você escolha mods específicos, além dos já pré determinados pelo Zona de Combate.
Mas atenção: o uso de qualquer mod que não seja os citados acima, impossibilitará sua entrada no servidor.

1- Clique no ícone Configurações.

2- Clique no botão Mods.

3- Clique na seta do External addons.

4- Selecione os mods desejados.
Em seguida, clique no botão Save.

5- Clique no botão Save para finalizar.
Pronto! Você já pode jogar com o seu mod favorito.
